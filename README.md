# Sistema Colaborador

Sistema de gestão de contatos com integração TSE para validação de zonas e sessões eleitorais.

## Tecnologias

- **Backend:** Node.js + Express + MySQL
- **Frontend:** React + Chart.js + Google Maps
- **Banco:** Supabase (PostgreSQL)

## Funcionalidades

- Cadastro de usuários com convites
- CRUD de contatos com dados eleitorais
- Dashboard com gráficos e estatísticas
- Mapa interativo com Google Maps
- Validação de zona/sessão com TSE
- Visualização em árvore (teia de relacionamentos)
